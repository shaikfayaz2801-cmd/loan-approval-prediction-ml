# 💰 Loan Approval Prediction

A machine learning application that predicts whether a loan application is likely to be approved based on applicant details.

## Features

- Loan approval prediction
- Decision Tree model
- Streamlit interface
- CSV dataset included

## Tech Stack

- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit

## Project Structure

```
loan_approval/
│
├── app.py
├── loan_approval_project.py
├── loan_approval_decision_tree_dataset.csv
```

## Installation

```bash
pip install pandas numpy scikit-learn streamlit
```

## Run

```bash
streamlit run app.py
```
